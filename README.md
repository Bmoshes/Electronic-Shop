# Electronic-Shop
A full-stack electronic shop web application built with React (client) and Node.js/Express (server).
```
my-react-store/
├── client/                # React frontend
│   ├── public/
│   ├── src/
│   │   ├── components/
│   │   │   ├── Navbar.jsx
│   │   │   ├── Home.jsx
│   │   │   ├── Product.jsx
│   │   │   ├── Cart.jsx
│   │   │   └── Order.jsx
│   │   ├── App.jsx
│   │   ├── index.jsx
│   │   └── api.js
│   ├── package.json
│   └── vite.config.js
└── server/                # Node.js backend
    ├── models/
    │   └── Product.js
    ├── routes/
    │   └── productRoutes.js
    ├── controllers/
    │   └── productController.js
    ├── config/
    │   └── db.js
    ├── server.js
    └── package.json
```
